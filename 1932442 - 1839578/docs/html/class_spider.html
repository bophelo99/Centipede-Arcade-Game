Detailed Description

This is a class that inherits from the IMovingEntity class. It models a Spider object in the game. It contains virtual functions that will be overridden. A Spider object moves in a zig-zag movement. This was achieved by calculating slopes in a random range based on the Spider's current position and moving it along the generated line. A Spider objects sets itself to dead if it reaches the grid's left or right boundaries.
