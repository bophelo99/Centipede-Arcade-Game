Detailed Description

Creates all the Mushroom objects of the game. The game grid is divided into cells of size 16x16 pixels. Each cell is paired with a bool indicating whether the cell is occupied or not. Mushrooms are randomly created on the grid excluding the player's screen boundary, they are created using the centre of each cell provided the cell is not occupied. Creation of one mushroom given a position only maps the position to a cell without querying it status.
