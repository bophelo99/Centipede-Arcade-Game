Detailed Description

This class is used to creates all the enemy objects of the game. It periodically creates Spider objects after a specified delay. It generates the initial Centipede train that is used in the beginning of the game. It can also generate a specified number CentipedeSegment of CentiBodyType::HEAD periodically.
