SplitAxis Class Reference

This is a class used to determine overlap between BoundaryBox objects. This class makes use of the Separating Axis Theorem algorithm that checks for overlap between convex polygons. The polygons implemented are Rectangle shapes which are created in the BoundaryBox class. The algorithm checks for overlap along all edges of the two BoundaryBox objects being checked. Projects the vertices of the BoundaryBox objects onto four axes. The axes are generated from the edges of the BoundaryBox objects, two from each one. The projections are then used to check for overlap. If a any of the axis detect no overlap, the algorithm returns false for no overlap. Otherwise, the other axis are checked for overlap. Detailed Description

This is a class used to determine overlap between BoundaryBox objects. This class makes use of the Separating Axis Theorem algorithm that checks for overlap between convex polygons. The polygons implemented are Rectangle shapes which are created in the BoundaryBox class. The algorithm checks for overlap along all edges of the two BoundaryBox objects being checked. Projects the vertices of the BoundaryBox objects onto four axes. The axes are generated from the edges of the BoundaryBox objects, two from each one. The projections are then used to check for overlap. If a any of the axis detect no overlap, the algorithm returns false for no overlap. Otherwise, the other axis are checked for overlap. The Minimum Translation Vector (MTV) is also calculated in order to move BoundaryBox objects out of overlap.
