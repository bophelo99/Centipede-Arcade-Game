Detailed Description

This is a class that inherits from the entityMovementBaseClass class. It models a Scorpion object in the game. It contains virtual functions that will be overridden. A Scorpion moves horizontally across the ScreenParameters. It sets itself to dead if it reaches the ScreenParameters's left or right boundaries.
